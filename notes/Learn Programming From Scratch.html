LEARN PROGRAMMING FROM SCRATCH

Kye Cunningham

Projects

The aim of these projects is to provide a foundation for study - not to provide a complete education on programming. Each one of these projects was chosen because they make use of different techniques that are commonly used throughout many other projects, and will hopefully provide a good starting point to making your own software of choice, whatever that may be.

Dungeons And Dragons Statsheet Tracker

Difficulty: 1

Languages: C++

Tools and Frameworks: MSVC

You will learn the basics of file IO in C++, and how to use stdin and stdout.

Text Adventure

Difficulty: 2

Languages: C++

Tools and Frameworks: MSVC

You will learn the basics of C++, as well as implement a basic state machine. You will then move on to saving your character to load later, as well as saving important world events.

Website

Difficulty: 3

Languages: HTML, CSS, Javascript

Tools and Frameworks: Jekyll, React, Node or Flask

You will first make a static website using HTML and CSS. Afterwards, you will gain experience with Javascript, before moving onto using React. Finally, you will use either Node.js or Flask to complete the project.
