Particle Editor

Introduction

Particle Editor is a scene editor that can be used to create scenes for Particle Engine. It was built with flexibility at the forefront of the architectural design. Since the editor is only used to create the scenes, speed isn’t the highest priority, and time can be taken to extend the toolset available to the development team instead of optimising for speed whilst restricting access to potentially incredible beneficial tools.

Particle Editor attempts to follow the C++ idiom of ‘pay for what you use’ - as well as allowing the editor to perform on lower-tier hardware, this also provides good scalability on higher-end hardware via plugins and optional environment settings.
